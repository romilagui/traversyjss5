Constructors

Constructor is a way of declaring a primitive value as an object. That is strings, numbers, booleans can be declared as primitive values or as objects.

Prototypes

Since ES5 does not use classes, object.prototype is used for inheritance.prototype is used when dealing with objects created through a constructor.

Prototypal Inheritance is used when one object type inherit from another. Call() is a function that allows us to call another function. Example: Person.call()
